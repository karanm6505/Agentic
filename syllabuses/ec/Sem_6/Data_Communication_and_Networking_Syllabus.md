# Course Syllabus
## Course Title: Data Communication and Networking
**Course Code:** CS340
**Credits:** 3
**Semester:** Fall 2024
**Instructor:** Dr. Anya Sharma
**Contact:** anya.sharma@university.edu

### Course Description:
This course provides a comprehensive exploration of advanced concepts in data communication and networking. Students will delve into the intricacies of network protocols, including TCP/IP, and gain a strong understanding of network security principles and practices. The course covers various network architectures, data transmission techniques, routing and switching mechanisms, and wireless communication technologies. Students will also gain practical experience through network design projects, hands-on labs, and case studies, culminating in a solid foundation for a career in networking.

### Prerequisites:
CS201 (Introduction to Computer Science), MATH160 (Calculus I)

### Learning Objectives:
1.  Analyze the layers of the TCP/IP model and their respective functionalities.
2.  Implement network protocols such as TCP, UDP, and HTTP.
3.  Evaluate different routing protocols and their performance characteristics.
4.  Design and configure network security measures, including firewalls and intrusion detection systems.
5.  Compare and contrast various wireless communication standards and their applications.
6.  Develop network design projects encompassing network topologies, addressing schemes, and security considerations.
7.  Apply network troubleshooting techniques to diagnose and resolve common network issues.

### Required Textbooks:
- Kurose, J. F., & Ross, K. W. *Computer Networking: A Top-Down Approach*, Pearson, 8th Edition, 2020 (ISBN: 978-0135928413)

### Recommended Readings:
- Tanenbaum, A. S., & Wetherall, D. J. "Computer Networks", Pearson, 6th Edition, 2021.
- "Network Security Bible", Eric Cole, Wiley, 2nd Edition, 2011.

### Weekly Schedule:
**Week 1: Introduction to Data Communication and Networking**
- Topics: Overview of networking concepts, network models (OSI, TCP/IP), network topologies, network devices.
- Readings: Kurose & Ross, Chapter 1
- Activities: Introduction to the course, discussion of networking fundamentals, lab setup.
- Due: N/A

**Week 2: Physical Layer and Data Transmission**
- Topics: Analog and digital signals, encoding techniques, transmission media (wired and wireless), data transmission impairments.
- Readings: Kurose & Ross, Chapter 2
- Activities: Lab: Cable testing and signal analysis.
- Due: Assignment 1: Signal Encoding problems.

**Week 3: Data Link Layer - Framing, Error Detection, and MAC Protocols**
- Topics: Framing, error detection and correction, MAC protocols (CSMA/CD, CSMA/CA), Ethernet.
- Readings: Kurose & Ross, Chapter 3
- Activities: Lab: Ethernet frame analysis.
- Due: Quiz 1: Covering Week 1-3 materials.

**Week 4: Data Link Layer - Ethernet and Wireless LANs**
- Topics: Ethernet, Wireless LANs (802.11 standards).
- Readings: Kurose & Ross, Chapter 3 & 5
- Activities: Lab: Setting up a wireless network.
- Due: N/A

**Week 5: Network Layer - IP Addressing and Subnetting**
- Topics: IP addressing, subnetting, CIDR.
- Readings: Kurose & Ross, Chapter 4
- Activities: Lab: IP addressing and subnetting exercises.
- Due: Assignment 2: IP addressing and subnetting problems.

**Week 6: Network Layer - Routing Protocols (RIP, OSPF)**
- Topics: Routing protocols (RIP, OSPF), routing algorithms.
- Readings: Kurose & Ross, Chapter 4
- Activities: Discussion: Comparison of routing protocols.
- Due: N/A

**Week 7: Transport Layer - TCP and UDP**
- Topics: TCP and UDP protocols, port numbers, connection management.
- Readings: Kurose & Ross, Chapter 3
- Activities: Lab: TCP and UDP programming.
- Due: Quiz 2: Covering Week 4-6 materials.

**Week 8: Transport Layer - Congestion Control and Flow Control**
- Topics: Congestion control, flow control, TCP performance.
- Readings: Kurose & Ross, Chapter 3
- Activities: Midterm Exam Review
- Due: N/A

**Week 9: Midterm Exam**
- Topics: Covers all material from weeks 1-8.
- Readings: Review all assigned readings.
- Activities: Midterm Exam.
- Due: Midterm Exam

**Week 10: Application Layer - DNS and HTTP**
- Topics: DNS and HTTP protocols.
- Readings: Kurose & Ross, Chapter 2
- Activities: Lab: DNS and HTTP client-server interactions.
- Due: N/A

**Week 11: Application Layer - FTP, SMTP, and Network Applications**
- Topics: FTP, SMTP, and other network applications.
- Readings: Kurose & Ross, Chapter 2
- Activities: Project Proposal Due
- Due: Project Proposal

**Week 12: Network Security - Cryptography and Authentication**
- Topics: Cryptography, authentication, and authorization.
- Readings: Kurose & Ross, Chapter 7
- Activities: Discussion: Security threats and vulnerabilities.
- Due: N/A

**Week 13: Network Security - Firewalls and Intrusion Detection**
- Topics: Firewalls, intrusion detection systems.
- Readings: Kurose & Ross, Chapter 7
- Activities: Lab: Firewall configuration.
- Due: Assignment 3: Network Security Design

**Week 14: Network Security - Network Security Protocols (SSL/TLS, IPSec)**
- Topics: Network security protocols (SSL/TLS, IPSec).
- Readings: Kurose & Ross, Chapter 7
- Activities: Presentation: Network Security research.
- Due: N/A

**Week 15: Wireless Networking and Emerging Technologies**
- Topics: Wireless standards (e.g., 802.11), wireless security, SDN, NFV.
- Readings: Kurose & Ross, Chapter 5 & relevant research papers.
- Activities: Student Presentations.
- Due: Project Presentations

**Week 16: Review and Project Presentations**
- Topics: Course Review, Project Presentations
- Readings: Review all course materials.
- Activities: Final Project Presentations and Course Wrap-up.
